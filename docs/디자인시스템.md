### Semantic Colors (아이콘/뱃지용)
- Pink: 플래시카드, 핵심 기능
- Green/Emerald: 성공, 설명
- Amber: 요약, 경고
- Violet: 퀴즈, 학습
- Blue: 정보, 표
- Teal: 예시, 보조

### 버튼 계층
1. **Primary**: primary 배경 + 흰 글씨 + shadow-lg shadow-{primary}/30
2. **Secondary**: primary 테두리 + primary 글씨 + 투명 배경 + hover시 채움
3. **Ghost**: 투명 + muted 글씨 + hover시 텍스트 강조

### 컴포넌트 스타일
- **카드**: rounded-2xl + border + bg-elevated
- **입력창**: rounded-xl + border + focus:border-primary
- **사이드바**: bg-elevated + border-r
- **아이콘 버튼**: w-10 h-10 + rounded-xl + semantic-color-bg + hover:scale-110

### 인터랙션
- 호버: border 색상 변경 + shadow-lg
- 아이콘 호버: scale-110 + transition-transform
- 버튼 클릭: scale-95 (active)
- 테마 전환: transition-colors duration-300
