# AWS RDS PostgreSQL 생성 가이드

## 접속

```
https://ap-northeast-2.console.aws.amazon.com/rds/
```

---

## 생성 단계

### 1️⃣ 데이터베이스 생성 시작
- **"데이터베이스 생성"** 버튼 클릭

---

### 2️⃣ 엔진 선택
- ✅ **PostgreSQL** 선택 (Aurora 선택 금지!)
- 버전: **PostgreSQL 15.x** (최신 안정 버전)

---

### 3️⃣ 템플릿
- ✅ **프리 티어** 선택

---

### 4️⃣ 설정

**DB 인스턴스 식별자**:
```
skn21-django-db
```

**마스터 사용자 이름**:
```
postgres
```

**마스터 암호**:
```
(강력한 비밀번호 설정 - 메모 필수!)
예: MyP@ssw0rd123!SKN
```

---

### 5️⃣ 인스턴스 구성
- DB 인스턴스 클래스: **db.t3.micro** (프리 티어)

---

### 6️⃣ 스토리지
- 할당된 스토리지: **20 GiB**
- 스토리지 유형: **범용 SSD (gp2)**
- ⚠️ **스토리지 자동 조정**: **비활성화** (비용 방지)

---

### 7️⃣ 연결 설정

**퍼블릭 액세스**:
- ✅ **예** 선택 (로컬에서 접근하려면 필수!)

**VPC 보안 그룹**:
- ✅ **"새로 생성"** 선택 (더 간단함)
- 새 VPC 보안 그룹 이름: 비워둬도 됨 (자동 생성)

---

### 8️⃣ 추가 구성

**데이터베이스 옵션**:
- 초기 데이터베이스 이름: `postgres` (또는 비워둠)

**백업**:
- ✅ **자동 백업 활성화**
- 백업 보존 기간: **7일** 권장

---

### 9️⃣ 데이터베이스 생성
- **"데이터베이스 생성"** 버튼 클릭
- ⏰ 대기 시간: **5-10분**

---

## 생성 완료 후 확인

### 필요한 정보 (메모하기!)

1. **엔드포인트**:
   ```
   skn21-django-db.xxxxx.ap-northeast-2.rds.amazonaws.com
   ```

2. **포트**:
   ```
   5432
   ```

3. **마스터 암호**:
   ```
   (설정한 비밀번호)
   ```
